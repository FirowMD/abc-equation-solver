# abc-equation-solver
A program that takes alphabetical equations like SEND+MORE=MONEY and finds their numerical solutions.

# Usage
Compile app with
```
gcc main.c
```
Run
```
./a.out <equation 1> [equation 2] ... [equation n]
./a.out SEND+MORE=MONEY
./a.out SEND+MORE=MONEY A+B+C=D
```
